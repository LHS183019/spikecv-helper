# spikecv-helper
A helper skill for AI Agents to interact with the [SpikeCV repository](https://github.com/Zyj061/SpikeCV), typically answering questions about SpikeCV and executing tasks for SpikeCV, an ultra-high-speed spike camera vision framework.

## When to use this Skill
+ Answering questions about SpikeCV's functionalities, algorithms, usage and community.
+ Tasks using the spikcv CLI, includes:
  + Downloading spike datasets 
  + Running processing spikecv algorithms (reconstruction, tracking, etc.) through the CLI interface.
